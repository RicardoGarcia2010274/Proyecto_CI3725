# Proyecto CI-3725, Fase 1

Esta primera etapa del proyecto corresponde al módulo de análisis lexicográfico del interpretador
del lenguaje BOT que queremos construir. Específicamente, se desea que Ud. implemente el referido

módulo utilizando una herramienta generadora de analizadores lexicográficos de las permitidas (men-
cionadas más adelante), y que además haga una breve revisión de los conceptos y métodos teórico-
prácticos relevantes.

El analizador lexicográfico a ser construido en esta etapa deberá aceptar como entrada cualquier
secuencia de caracteres y dar como salida los tokens relevantes reconocidos. Si se reciben caracteres
que no corresponden a ningún token del lenguaje BOT, debe mostrarse un mensaje de error. Cada
token debe ir acompañado de su número de fila y de columna.
